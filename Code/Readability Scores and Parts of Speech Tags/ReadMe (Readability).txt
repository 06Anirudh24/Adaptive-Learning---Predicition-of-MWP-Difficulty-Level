1) Gunning Fog (GF) Index - Estimates number of years of formal education reqd to understand the text. (Lower score => easy to read). 
Fromula: 0.4[ (words/sentence) + 100(complex words/words) ]
Reference: https://en.wikipedia.org/wiki/Gunning_fog_index

2) Flesch - Kincaid (FK) Readability Index  (High score => Easy to read)
Formula: 206.835 - 1.015(total words/total sentences) - 84.6(total syllables/total words)
Reference: https://en.wikipedia.org/wiki/Flesch%E2%80%93Kincaid_readability_tests

3) SMOG (Simple Measure of Gobbledygook) Index -  Estimates number of years of formal education reqd to understand the text. (Lower score => easy to read). 
Fromula: 3.1291 + 1.0430(sqrt(number of polysyllables * 30 /number of sentences))
Reference: https://en.wikipedia.org/wiki/SMOG
Polysyllables => Words of >=3 syllables 

4) Automated Readability (AR) Index (Used in Prev Sem Work) (Lower score => easy to read).
Fromula: 4.71(characters/word) + 0.5(words/sentence) - 21.43
Reference: https://en.wikipedia.org/wiki/Automated_readability_index

5) Coleman-Liau (CL) Index - Estimates understandbility of a text (Lower score => easy to read. Can take negative values also). 
Formula: 0.0588(Avg characters per 100 words) - 15.8 - 0.296(average number of sentences per 100 words)
Reference: https://en.wikipedia.org/wiki/Coleman%E2%80%93Liau_index
